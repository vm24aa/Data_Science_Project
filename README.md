This project aims to detect Parkinson’s Disease using machine learning techniques applied to speech acoustic features. Parkinson’s disease affects motor control, which can cause noticeable changes in speech patterns such as instability in pitch, loudness, and vocal vibrations. By analyzing these speech characteristics, machine learning models can help identify potential indicators of the disease.

The dataset used in this project contains 756 speech samples with 755 extracted acoustic features. The workflow includes exploratory data analysis, data preprocessing, feature selection, and training classification models.

Two ensemble learning algorithms, Random Forest and XGBoost were implemented to classify whether a subject is healthy or has Parkinson’s disease. The models were evaluated using metrics such as accuracy, confusion matrix, and classification report.

This project demonstrates how machine learning can be applied to biomedical data to support early detection and analysis of neurological disorders through speech analysis.
